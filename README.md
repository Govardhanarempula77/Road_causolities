# 🚧 Highway Data Analysis & GIS Project

## 📌 Overview
This project demonstrates data analysis and GIS skills applied to real-world highway and transport datasets. It focuses on analysing road conditions, traffic, and safety data to support decision-making for infrastructure management.

The project is inspired by real-world use cases in highway asset management, where data is used to prioritise maintenance and improve road safety.

---

## 🎯 Objectives
- Clean and preprocess raw datasets
- Perform exploratory data analysis (EDA)
- Analyse road condition and traffic patterns
- Integrate spatial (GIS) data
- Identify high-risk or high-priority road segments
- Present insights in a clear and actionable way

---

## 📊 Datasets Used
- UK Road Safety Data (accidents)
- OpenStreetMap road network data
- (Optional) Traffic count or road condition datasets

---

## 🛠️ Tools & Technologies
- **Python** (pandas, numpy)
- **SQL** (data querying and aggregation)
- **GIS** (GeoPandas / QGIS)
- **Jupyter Notebook**
- **Data visualisation** (matplotlib / seaborn)

---

## 🔍 Key Tasks Performed

### 1. Data Cleaning
- Handled missing values
- Converted data types
- Removed duplicates
- Validated dataset integrity

### 2. Data Analysis
- Calculated average road condition by region
- Analysed traffic patterns
- Identified low-condition (high-risk) roads

### 3. Spatial Analysis (GIS)
- Loaded and processed geospatial data
- Performed spatial joins (e.g. accidents to roads)
- Created buffers to analyse proximity
- Identified accident hotspots

### 4. Insights & Findings
- Roads with high traffic and low condition scores require urgent attention
- Certain regions show higher accident density
- Spatial clustering highlights high-risk zones
